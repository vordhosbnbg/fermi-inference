## Command

```bash
./build/llama.cpp-opencl-native/bin/llama-cli \
  --device none \
  -ngl 0 \
  -m models/Qwen3-0.6B-Q4_0.gguf \
  -p 'Answer in one sentence: what is OpenCL?' \
  -c 256 \
  -n 32 \
  -t 4 \
  --no-display-prompt \
  --single-turn \
  --reasoning off
```

## Notes

- The command was CPU-only: `--device none -ngl 0`.
- Inside the Codex sandbox, OpenCL platform discovery still reports no platform:
  `ggml_opencl: platform IDs not available.`
- `llama-cli --no-conversation` is not supported in this build; use
  `--single-turn` for a one-shot chat completion.
